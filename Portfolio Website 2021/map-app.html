MaptyMap App

running and cycling logging app

A small app created with runners in mind. This app works with an API called leaflet, to allow the user to log running or cycling workouts. The user can save information like distance, duration, and cadence and switch between running type workouts or cycling type workouts. The real cool part about this is that the user can pick anywhere on a map and place a pin that holds the running information and the date of that specific instance. All done using HTML, CSS, and JavaScript.
